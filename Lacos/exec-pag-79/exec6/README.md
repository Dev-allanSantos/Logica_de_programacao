# Enunciado
Faça um programa que leia uma quantidade desconhecida de números e conte quantos deles estão nos seguintes intervalos: 
* [0 – 25.9] 
* [26 – 50.9]  
* [51 – 75.9]  
* [76 – 100]. 

A entrada de dados deve terminar quando for lido um número negativo.
